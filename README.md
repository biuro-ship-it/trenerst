Asystent Strzelecki - Inteligentny System Analizy TreninguAsystent Strzelecki to aplikacja stworzona dla strzelców sportowych i rekreacyjnych, która automatyzuje proces analizy tarcz, śledzi postępy oraz pomaga w zarządzaniu logistyką sprzętową.🎯 Kluczowe FunkcjonalnościAnaliza Obrazu (OpenCV): Automatyczne wykrywanie przestrzelin na zdjęciach tarczy przy użyciu zaawansowanego przetwarzania konturów.Wirtualny Trener: Natychmiastowe porady po oddanej serii (np. wykrywanie zrywania spustu czy oczekiwania na odrzut) na podstawie rozkładu przestrzelin.Logistyka i Koszty: Baza danych broni i amunicji z automatycznym kalkulatorem kosztów pojedynczego strzału oraz całej sesji.Statystyki Skupienia: Precyzyjne obliczanie rozmiaru grupy w milimetrach oraz minutach kątowych (MOA).Archiwizacja: Przechowywanie historii sesji w lokalnej bazie danych SQLite.🚀 TechnologieJęzyk: Python 3.14+Biblioteki: OpenCV, NumPy, SQLite3Baza danych: SQLite🛠️ Instalacja i UruchomienieSklonuj repozytorium:git clone [https://github.com/biuro-ship-it/trenerst.git](https://github.com/biuro-ship-it/trenerst.git)
cd trenerst
Zainstaluj wymagane biblioteki:pip install opencv-python numpy
Uruchom aplikację:python shooting_assistant_backend.py
